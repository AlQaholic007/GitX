# GitX

GitX is a cross-platform desktop application for managing code on GitHub

# Features
 - Clean delivery of github
 - Auto updates to the latest github release
 - Super easy to use
 - Clean UI

 [Downloads](https://github.com/AlQaholic007/GitX/releases)

### Supported devices
 - MacOS
 - Windows
 - Linux
 - Android
